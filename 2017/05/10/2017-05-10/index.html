tech learning

git shallow clone

链接地址

因为 git clone 不仅下载文件，还会下载每一个版本变更。在持续集成（CI）、持续交付（CD）中，自动化的下载容易引起 git 服务器负载问题。。

git clone --depth 1 能简单地避免该问题。

Git 1.9 版本之前，shallow clone 的代码库不支持 pull/push。
